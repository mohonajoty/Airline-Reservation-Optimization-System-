# ✈ Airline Reservation & Optimization System

## 📌 Project Description
The Airline Reservation & Optimization System is a console-based application developed using the C programming language.  

This project simulates a basic airline management system where users can search flights, book tickets, process payments, and manage passenger records.

In addition to standard reservation features, the system also integrates important graph and greedy algorithms such as:
- Dijkstra’s Algorithm (Shortest Path)
- Prim’s Algorithm (Minimum Spanning Tree)
- Ford-Fulkerson Algorithm (Maximum Flow)

These algorithms help in solving real-world problems like route optimization, cost reduction, and passenger flow analysis.

---

## 🎯 Project Objective
The main objective of this project is to understand how real-world airline systems work and how computer algorithms can improve efficiency in transportation systems.

It helps to:
- Manage airline operations efficiently
- Reduce manual work
- Improve decision-making using algorithms
- Provide a structured and automated system

## ⚙ How It Works
1. User logs into the system (optional authentication module)
2. System displays available flights
3. User selects a flight and enters passenger details
4. Payment is simulated (Card / Mobile / Cash)
5. Ticket is generated automatically
6. Passenger data is stored in system memory
7. Optimization algorithms help analyze routes and network efficiency

---

## 📊 Algorithms Used
- **Dijkstra Algorithm:** Finds the shortest path between cities  
- **Prim’s Algorithm:** Builds minimum cost network for flights  
- **Ford-Fulkerson Algorithm:** Calculates maximum passenger flow between routes  
- **Greedy Method:** Used in seat allocation and cheapest flight selection
